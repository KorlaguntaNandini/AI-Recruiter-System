# 🧠 AI Recruiter System

An Explainable AI-powered resume ranking system that analyzes resumes and job descriptions using semantic search and machine learning.

---

## 🚀 Features

- Semantic Resume Matching (not keyword-based)
- Explainable AI (shows why candidate is selected/rejected)
- Skill Extraction (Python, ML, SQL, etc.)
- Skill Gap Analysis (why NOT selected)
- Confidence Score (HIGH / MEDIUM / LOW)
- Streamlit Web UI
- Excel export of ranked candidates

---

## 🏗️ Tech Stack

- Python
- Streamlit
- Sentence Transformers (MiniLM model)
- Scikit-learn
- PyPDF
- OpenPyXL
